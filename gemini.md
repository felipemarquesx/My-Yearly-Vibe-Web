# Projeto AV2 - Desenvolvimento de Software

## Objetivo

Desenvolver um Proof of Concept (POC) ou Minimum Viable Product (MVP) utilizando React, Docker e arquitetura em 3 camadas ou MVC.

O projeto deve priorizar organização, clareza técnica, boas práticas de desenvolvimento e facilidade de apresentação.

---

## Requisitos Obrigatórios

### Equipe
- Equipe com no máximo 10 integrantes.

### Frontend
- Utilizar React.
- Utilizar um framework CSS:
  - Tailwind CSS
  - Bootstrap
  - Ou equivalente.

### Backend
- Seguir arquitetura:
  - MVC (Model-View-Controller)
  - OU Arquitetura em 3 Camadas.

### Infraestrutura
- Utilizar Docker para execução da aplicação.
- O ambiente deve ser reproduzível através de containers.

### Dados
- Pode utilizar:
  - API pública;
  - Dados mockados;
  - Banco de dados local.

### Controle de Versão
- Utilizar Git e GitHub.
- Trabalhar obrigatoriamente com colaboração entre membros.
- Seguir o padrão de commits definido pela equipe.

---

## Padrão de Commits

Utilizar o formato:

- feat: nova funcionalidade
- fix: correção de erro
- refactor: refatoração de código
- style: ajustes visuais
- docs: documentação
- test: testes
- chore: tarefas gerais

Exemplos:

```bash
feat: adicionar tela de login
fix: corrigir validação de formulário
docs: atualizar documentação da API
